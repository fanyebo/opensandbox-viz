# opensandbox-viz

Streamlit dashboard for OpenSandbox sandbox management.

## Rules
- 代码不改不 commit，浏览器测试→用户确认→同意后提交
- 问题报告格式：原因→方案→结果
- 修改前先 codegraph_explore（CodeGraph 已索引）

## Stack
Streamlit + httpx + PyInstaller. Port 8501.

## Commands
```bash
uv run streamlit run app.py      # 开发
uv run pyinstaller run.spec      # 打包
```
